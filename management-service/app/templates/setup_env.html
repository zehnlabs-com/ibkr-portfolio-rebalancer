Configuration Settings

Edit your environment variables below. The system will load your existing .env file if it exists, otherwise it will show the template from .env.example.

Important Settings to Configure:

  • IB_USERNAME and IB_PASSWORD - Your Interactive Brokers login credentials
  • ALLOCATIONS_API_KEY and REBALANCE_EVENT_SUBSCRIPTION_API_KEY - Your Zehnlabs API keys
  • TRADING_MODE - Set to "live" for real trading or "paper" for testing
  • USER_NOTIFICATIONS_CHANNEL_PREFIX - Unique identifier for your notifications
